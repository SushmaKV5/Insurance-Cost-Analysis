# 🏥 Insurance Charge Analysis

## 📊 Project Overview
This project performs Exploratory Data Analysis (EDA) on a medical insurance dataset to understand the factors that influence insurance charges. The goal is to extract meaningful insights from the data and identify patterns related to demographics and lifestyle.

---

## 🎯 Objectives
- Analyze the relationship between insurance charges and features like age, BMI, smoking status, and region
- Identify key factors that significantly impact medical costs
- Visualize trends and patterns using graphs

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📁 Dataset Features
The dataset includes:
- **Age** – Age of the individual
- **Sex** – Gender
- **BMI** – Body Mass Index
- **Children** – Number of dependents
- **Smoker** – Smoking status
- **Region** – Residential area
- **Charges** – Medical insurance cost

---

## 📌 Key Insights
- 🔥 Smoking significantly increases insurance charges
- 📈 Charges tend to increase with age
- ⚖️ Higher BMI is associated with higher medical costs
- 🌍 Regional differences exist but are less impactful than lifestyle factors

---

## 📊 Visualizations Included
- Distribution plots
- Correlation heatmap
- Box plots for categorical comparisons
- Scatter plots for trend analysis
